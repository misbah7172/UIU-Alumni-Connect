# Build a Full Alumni Community & Career Networking Platform

You are a senior full-stack software architect and AI coding agent.

Your task is to design and develop a production-ready web platform called:

# “UIU Alumni Connect”

A modern alumni-student networking platform for university communities where alumni can post jobs, mentor students, answer questions, and provide referrals.

The platform must be scalable, secure, modular, responsive, and designed using modern industry standards.

---

# Primary Goal

Create a professional platform where:

* Alumni can post jobs, internships, referrals, mentorship opportunities, and answer student questions.
* Students can create profiles, ask career questions, apply to jobs, request referrals, and connect with alumni.
* Admins can verify users and moderate the platform.

The platform should function like a combination of:

* LinkedIn
* Stack Overflow
* Reddit
* University Alumni Portal
