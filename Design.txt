Create a modern, professional, responsive university alumni networking platform UI/UX design for a web application named “UIU Alumni Connect”.

The design must feel:

* Professional
* Academic
* Trustworthy
* Clean
* Modern
* Fast
* Minimal but premium
* Similar to a combination of LinkedIn, Notion, GitHub, and modern university portals

The platform is designed for:

* University students
* Alumni
* Mentors
* Recruiters
* Administrators

The overall visual identity should strongly reflect:

* University culture
* Professional networking
* Career growth
* Collaboration
* Community building

Use a premium modern SaaS-style interface with university branding aesthetics.

---

# Design Goals

The website must:

* Feel enterprise-grade
* Be extremely responsive
* Load very fast
* Have smooth interactions
* Support all screen sizes
* Be optimized for accessibility
* Maintain clean information hierarchy
* Use minimal but elegant animations

The UI must be:

* Mobile-first
* Tablet optimized
* Desktop optimized
* Touch-friendly
* Highly readable

---

# Design Style

Use:

* Modern card-based layouts
* Soft shadows
* Rounded corners
* Glassmorphism only subtly
* Spacious layouts
* Consistent spacing system
* Elegant typography
* Professional dashboard design
* Smooth hover effects
* Sticky navigation
* Clean sidebar layouts

Avoid:

* Overcrowded UI
* Excessive colors
* Heavy gradients
* Complex animations
* Cluttered dashboards

---

Primary:

* Professional gold

* Be highly readable

# Animation Requirements

Use subtle animations only:

* Hover transitions
* Fade-ins
* Card elevation
* Smooth page transitions
* Notification animations

Avoid:

* Heavy motion
* Distracting effects
* Slow animations

Use Framer Motion for modern interactions.
